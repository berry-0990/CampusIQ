# CampusIQ — A RAG-Based Campus Knowledge Assistant

CampusIQ is an AI-powered educational assistant designed to reduce hallucinations in Large Language Models using Retrieval-Augmented Generation (RAG).

## Features
- Semantic Search
- Source-Cited Responses
- Hallucination Reduction
- Vector Database Retrieval
- Educational AI Assistance

## Tech Stack
- LangChain
- ChromaDB
- GPT-4o-mini
- Mistral-7B
- BAAI Embeddings
